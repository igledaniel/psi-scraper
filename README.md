# psi-scraper
The Ion Cannon for scraper, with json configs, proxy, Gulp build system, parallelization, Sequelize data model.


## Installation

1. clone this repo
2. install dependencies with `npm run setup` (this runs `npm i` too)
3. setup `config/`
4. run with `npm start`


## Development

#### Test

```shell
npm test
```

#### ESLint

```shell
npm run lint
```
